 Hospital Patient Management System (HPMS)

The **Hospital Patient Management System** is a database project designed to manage patient records, doctor details, appointment scheduling, and medical report storage.  
This system helps hospitals automate administrative tasks, reduce manual errors, and provide fast access to patient medical history.
